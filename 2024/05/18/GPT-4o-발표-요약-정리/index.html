    GPT-4o 발표 보고서

    주요내용

    • GPT-4o는 텍스트, 오디오, 이미지 입력을 동시에 받아들이고, 이러한 출력도 동시에 생성 가능
    • GPT-4 Turbo 대비 비영어 텍스트에서 성능 향상 및 처리 속도 증가, API 가격 50% 절감
    • 실시간 반응 속도 232ms로 사람과 유사, 비전 및 오디오 이해 능력 크게 향상
    • 다국어 토큰화 효율성 향상: 예를 들어, 한국어는 1.7배, 중국어는 1.4배, 일본어는 1.4배 효율적으로 토큰화

    상세내용

    GPT-4o 모델 성능

    • GPT-4 Turbo와 텍스트, 추론, 코드 작성 능력에서 동등한 성능
    • 다국어, 오디오, 비전 기능에서 새로운 최고 성능 달성
    • 실시간 대화 반응 속도: 최소 232ms, 평균 320ms
    • API 비용: 입력 $5.00/1M 토큰, 출력 $15.00/1M 토큰
    • GPT-4 Turbo 대비 2배 빠르고, API 비용 50% 절감

    그림 1: 모델 성능 비교

    모델 텍스트 성능 다국어 성능 오디오 성능 비전 성능 반응 속도 가격(입력/출력)
    GPT-4o 동등 최고 최고 최고 최소 232ms $5.00 / $15.00
    GPT-4 Turbo 동등 높음 높음 높음 2.8-5.4초 $10.00 / $30.00

    다국어 토큰화 효율성

    • 다국어 텍스트 토큰화 효율성 증가
    • 구자라티어: 145 -> 33 (4.4배 효율 증가)
    • 텔루구어: 159 -> 45 (3.5배 효율 증가)
    • 타밀어: 116 -> 35 (3.3배 효율 증가)
    • 한국어: 45 -> 27 (1.7배 효율 증가)
    • 중국어: 34 -> 24 (1.4배 효율 증가)
    • 일본어: 37 -> 26 (1.4배 효율 증가)

    표 1: 다국어 토큰화 효율성

    언어 기존 토큰 수 새 토큰 수 효율성 증가 비율
    구자라티어 145 33 4.4x
    텔루구어 159 45 3.5x
    타밀어 116 35 3.3x
    한국어 45 27 1.7x
    중국어 34 24 1.4x
    일본어 37 26 1.4x

    한국어를 포함한 아시아권 언어 성능

    • 한국어: 기존 모델 대비 1.7배 토큰 효율성 향상, 텍스트 생성 및 이해 능력 대폭 향상
    • 중국어: 토큰 효율성 1.4배 향상, 자연스러운 문맥 이해와 생성 능력 강화
    • 일본어: 토큰 효율성 1.4배 향상, 복잡한 문법 구조 이해 및 텍스트 생성 능력 개선
    • 베트남어, 힌디어, 우르두어 등 다양한 아시아권 언어에서도 성능 향상

    모델 예시

    • 두 개의 GPT-4o가 상호작용하며 노래를 부르는 장면
    • 면접 준비: 실제 면접 상황을 시뮬레이션하여 사용자 지원
    • 가위바위보: 실시간으로 사용자와 가위바위보 게임 진행
    • 실시간 번역: 다양한 언어를 실시간으로 번역하여 제공
    • 아빠 농담: 사용자와 재미있는 농담을 주고받는 기능
    • 안녕하세요 및 여러 가지 인사말: 다양한 언어로 사용자에게 인사

    모델 안전성과 한계

    • 다양한 안전성 평가를 통과하여 중간 위험 수준 이하로 평가됨
    • 음성 출력에서 새로운 안전 시스템 적용
    • 외부 전문가들과의 협업을 통해 모델의 리스크 식별 및 안전성 강화
    • 오디오 모달리티는 제한된 사전 설정된 목소리로 출력, 추가적인 안전 시스템 적용 예정

    그림 2: 모델 안전성 평가 기준

    평가 항목 위험 수준
    사이버 보안 중간 이하
    설득력 중간 이하
    모델 자율성 중간 이하
    심리적 영향 중간 이하

    모델 한계

    • 모든 모달리티에서 몇 가지 제한 사항 존재
    • 텍스트 및 이미지 입력과 출력은 즉시 사용 가능하나, 오디오 출력은 사전 설정된 목소리만 사용 가능
    • 모델의 성능을 지속적으로 개선하기 위한 피드백 수집 중

    모델 이용 가능성

    • GPT-4o는 실용성을 염두에 두고 개발된 최신 다중 모달 모델
    • ChatGPT 무료 계층에서 사용 가능, Plus 사용자에게는 최대 5배 더 많은 메시지 제한 제공
    • 개발자는 API를 통해 텍스트 및 비전 모델로 접근 가능
    • API 가격: GPT-4o는 GPT-4 Turbo 대비 절반 가격, 2배 빠른 처리 속도

    그림 3: 모델 가격 비교

    GPT-4 Turbo 10.00 30.00
