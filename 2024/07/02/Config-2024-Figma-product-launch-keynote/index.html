    서론

    행사: Config 2024
    연사: 딜런 필드 (Figma 공동 창립자 및 CEO)

    Config 2024에서 Figma는 혁신적인 AI 기능을 중심으로 다양한 업데이트를 발표했습니다. Figma AI는 사용자가 텍스트 설명으로 디자인을 생성할 수 있는 ‘Make Design’ 기능, 유사한 요소를 빠르게 찾을 수 있는 ‘Search for Similar’ 기능, 그리고 작업 흐름을 유지하도록 돕는 AI 자동 생성 기능을 포함하고 있습니다. 또한, 이미지 배경 제거, 다국어 번역, 레이어 명 자동 정리, 프로토타입 자동 생성 등 AI를 활용한 다양한 기능을 통해 디자인 과정을 혁신적으로 변화시킵니다. Figma AI는 작업 효율성을 극대화하고, 디자인과 개발 간의 협업을 더욱 원활하게 만들어 줍니다. 이번 업데이트는 AI를 통해 사용자 경험을 크게 개선하고, 디자인 작업을 보다 효율적으로 수행할 수 있도록 돕습니다.

    주요 발표 내용

    UI3 도입

    Figma는 새로운 사용자 인터페이스(UI3)를 도입하여 디자인 경험을 개선했습니다.

    • 새 UI3: 간결하고 정돈된 인터페이스로, 도구 모음이 하단으로 이동하여 콘텐츠를 중심에 두도록 설계되었습니다.
    • 피드백 반영: 사용자 피드백을 반영하여 인터페이스를 점진적으로 개선하고, 쉽게 전환할 수 있는 기능을 제공합니다.

    Auto Layout

    Auto Layout 기능이 더욱 직관적으로 개선되었습니다.

    • 자동 레이아웃: 사용자가 간편하게 접근하고 활용할 수 있도록 ‘Suggest Auto Layout’ 기능이 추가되었습니다.
    • 자동 조정: 다양한 화면 크기에 맞게 디자인 요소가 자동으로 조정됩니다.

    개발자 모드 업데이트

    개발자 모드가 크게 향상되어 디자이너와 개발자의 협업을 더욱 원활하게 합니다.

    • Dev 모드: 대규모 파일을 쉽게 검사하고, 변경 사항을 효율적으로 관리할 수 있는 새로운 상태 표시 기능이 추가되었습니다.
    • 코드 연결: Code Connect 기능을 통해 디자인 시스템과 실시간으로 코드를 연동할 수 있습니다.

    Code Connect

    Code Connect를 통해 디자인 시스템을 Dev 모드에서 직접 활용할 수 있습니다.

    • 실시간 코드 업데이트: 디자인 변경 사항이 즉시 코드에 반영되어 시간과 노력을 절약합니다.
    • 협업 사례: Spotify와의 협업 사례를 통해 Code Connect의 효율성을 입증하였습니다.

    Figma AI

    Figma AI는 사용자의 작업 효율성을 높이고 디자인 과정을 혁신적으로 변화시킬 다양한 기능을 제공합니다.

    디자인 생성 (Make Design)

    기능 설명:

    • Prompt 기반 디자인 생성: 사용자가 텍스트로 입력한 설명에 따라 디자인을 자동으로 생성합니다.
    • 테마 및 색상 조정: 생성된 디자인의 테마와 색상을 쉽게 변경할 수 있습니다.

    예시:

    • “피자 찾기 앱”과 같은 설명을 입력하면 AI가 해당 디자인을 생성합니다.
    • 테마를 변경하거나 색상을 조정하여 맞춤화할 수 있습니다.

    필요한 요소 검색 (Search for Similar)

    기능 설명:

    • 유사한 요소 검색: 디자인 요소를 빠르게 검색하여 필요한 요소를 찾을 수 있습니다.
    • 이미지 및 드로잉 검색: 이미지나 드로잉을 통해 유사한 디자인 요소를 검색할 수 있습니다.

    예시:

    • 스크린샷을 통해 해당 디자인 파일을 검색하거나, 간단한 드로잉을 통해 필요한 아이콘을 찾을 수 있습니다.

    작업 흐름 유지 (Stay in the Flow)

    기능 설명:

    • AI 자동 생성: AI가 자동으로 디자인 요소를 추가하거나 수정하여 작업 흐름을 유지합니다.
    • 배경 제거 및 다국어 번역: 이미지의 배경을 제거하거나 텍스트를 여러 언어로 번역할 수 있습니다.

    예시:

    • “Make Image” 기능을 통해 원하는 이미지를 생성하고, “Rewrite” 기능을 통해 텍스트를 다시 작성하거나 번역할 수 있습니다.
    • “Name Layers” 기능을 통해 레이어 명을 자동으로 정리할 수 있습니다.

    Figma 슬라이드 도입

    Figma는 슬라이드 제작 기능인 Figma Slides를 도입하여 사용자가 프레젠테이션을 쉽게 만들고 협업할 수 있도록 지원합니다.

    • Figma Slides: 슬라이드를 제작하고 협업할 수 있는 기능을 제공합니다.
    • 그리드 모드: 슬라이드의 전체 구조를 한눈에 볼 수 있는 그리드 모드.
    • AI 텍스트 톤 조절: 텍스트 톤을 조절하여 내용을 간결하게 또는 자세하게 수정.
    • 인터랙티브 디자인: 프로토타입 링크를 삽입하여 슬라이드에서 직접 인터랙티브한 디자인 사용 가능.

    추가 업데이트

    1. 데스크탑 앱: 새로운 탭 페이지 및 파일 탐색기 경험 개선.
    2. FigJam: 여러 페이지를 추가할 수 있는 기능 도입.

    결론

    Figma의 새로운 AI 기능과 UI3, Dev 모드 업데이트는 사용자 작업 효율성을 높이고, 디자인과 개발 간의 협업을 촉진하는 데 중점을 두고 있습니다. 또한, Figma Slides 기능을 통해 사용자는 슬라이드 프레젠테이션을 쉽게 만들고, 실시간으로 협업할 수 있습니다. AI를 활용한 다양한 기능은 디자인 과정에서 발생하는 문제를 해결하고, 작업 효율성을 극대화하는 데 큰 도움이 될 것입니다. Figma는 앞으로도 사용자 피드백을 반영하여 지속적으로 발전해 나갈 것입니다.
