    • 2024년 7월 26일 AI 소식

    OpenAI에서는 SearchGPT 프로토타입을 출시하여 사용자가 웹에서 빠르고 신뢰할 수 있는 답변을 쉽게 찾을 수 있도록 하였습니다. 이 프로토타입은 소수의 사용자 및 출판사 그룹에게 제공되어 피드백을 수집하고 있습니다. 또한, Rule-Based Rewards(RBR)을 이용해 모델의 안전성을 향상시키는 새로운 방법을 개발하였으며, 이 방법은 인간의 피드백을 대체하여 효율성을 높이고 있습니다. META에서는 Llama 3.1과 관련된 새로운 리소스를 GitHub에 업데이트했으며, Llama Guard 3와 Prompt Guard와 같은 새로운 신뢰 및 안전 도구를 공개하였습니다. Mistral AI는 1230억 개의 매개변수를 가진 대형 모델 Mistral Large를 출시하였고, Hugging Face에서 사용할 수 있게 되었습니다. PyTorch는 2.4 버전을 출시하여 최신 Python 3.12 지원, AOTInductor 성능 최적화 등 다양한 기능을 추가하였습니다. Open Sora는 3D 비디오 확산 모델을 선보이며, InternVL2-Pro는 비전-언어 처리에 있어 GPT-4V, Claude 3.5 Sonnet, Gemini Pro 1.5 를 뛰어넘는 우수한 성능을 보였습니다.

    OpenAI, SearchGPT 프로토타입 출시

    링크, 2024년 7월 25일

    • 새로운 AI 검색 기능을 테스트하는 SearchGPT 프로토타입 출시
    • 소규모 사용자 및 출판사 그룹에게 제공하여 피드백 수집 중
    • 대화형 AI 모델과 실시간 웹 정보를 결합하여 빠르고 신뢰할 수 있는 답변 제공
    • 사용자가 후속 질문을 할 수 있어 대화형 검색 경험 제공
    • 출판사와 협력하여 사용자가 고품질 콘텐츠를 쉽게 찾을 수 있도록 지원
    • 검색 결과에 명확한 인라인 인용 및 링크 제공
    • 출판사 관리 도구 제공, 검색 결과 노출 방식 제어 가능
    • 프로토타입에서 얻은 피드백을 기반으로 향후 ChatGPT에 통합 예정

    OpenAI, Rule-Based Rewards를 통한 모델 안전성 향상

    링크, 2024년 7월 24일

    • Rule-Based Rewards(RBR)을 이용해 모델의 안전성을 높이는 새로운 방법 개발
    • 인간의 피드백 수집 없이 효율적으로 안전한 모델 행동 유도
    • 안전 정책 변화 시 새로운 데이터 없이도 규칙만 수정하여 빠르게 대응 가능
    • RBR은 GPT-4 출시 이후 안전 스택의 일부로 사용됨
    • 단순한 규칙을 정의하여 모델의 출력이 안전 기준을 충족하는지 평가
    • 다양한 유해하거나 민감한 주제에 대해 세 가지 이상적인 모델 행동 정의
    • RBR을 활용한 모델은 안전성과 유용성 간의 균형을 유지함
    • 인간 피드백과 결합하여 주관적인 작업에도 적용 가능

    META, Llama 3.1 리소스 업데이트

    링크, 2024년 7월 25일

    • Llama 3.1 관련 새로운 코드 및 학습 레시피 GitHub에 업데이트
    • 최신 신뢰 및 안전 도구, 새로운 참조 시스템 등 포함
    • 에이전틱 작업 수행 가능한 시스템으로 Llama 3.1 운영
    • Llama Guard를 통해 입력 및 출력 필터링 지원
    • 예제 스크립트와 노트북 제공, 다양한 사용 사례에 신속히 적용 가능

    META, Llama Guard 3 및 Prompt Guard 출시

    링크, 2024년 7월 25일

    • Llama Guard 3, 최신 LLM 애플리케이션 모더레이션 도구 출시
    • 8개 언어 지원, 긴 컨텍스트 윈도우 프롬프트 지원
    • Llama 3.1 8B 베이스 모델에 훈련됨
    • 유연한 개발자 콘텐츠 표준에 맞게 조정 가능
    • Prompt Guard, 악성 프롬프트 공격 방지 도구 출시
    • 멀티링구얼 베이스 모델, 비용 효율적인 필터링 제공

    Mistral, 대형 모델 Mistral Large 출시

    링크, 2024년 7월 25일

    • 1230억 개의 매개변수를 가진 Instruct 모델 출시
    • 128k 컨텍스트 지원, 다중 언어 지원
    • 80개 이상의 코딩 언어 학습
    • 도구 사용 및 함수 호출 지원
    • Hugging Face에서 비상업적 연구 목적으로 사용 가능
    • MMLU 84%, HumanEval 92% 등 높은 성능 달성

    Team PyTorch, PyTorch 2.4 출시

    링크, 2024년 7월 24일

    • Python 3.12 지원, AOTInductor 성능 최적화 등 다양한 기능 추가
    • 새로운 TCPStore 서버 백엔드 도입, 초기화 시간 단축
    • 새로운 Python Custom Operator API, 커스텀 커널 통합 용이
    • FSDP2, torch.distributed.pipelining 등의 프로토타입 기능 추가
    • AWS Graviton 최적화, BF16 심볼릭 셰이프 최적화 등 성능 향상

    Open Sora, Open Sora Plan v1.2.0 출시

    링크, 2024년 7월 25일

    • 3D 비디오 확산 모델 v1.2.0 출시
    • 3D 전체 주의 아키텍처 사용, 4초 720p 비디오 학습
    • Huawei Ascend AI 컴퓨팅 시스템에서 학습 가능
    • GitHub에서 오픈 소스로 제공, 기여 환영

    OpenGVLab, InternVL2-Pro 출시

    링크, 2024년 7월 25일

    • InternVL2-Pro는 OpenGVLab에서 개발한 다중 모드 AI 모델
    • MMMU 벤치마크에서 62%의 정확도로 GPT-4o와 동일한 성능을 보임
    • DocVQA와 InfoVQA 벤치마크에서 최고 성능(SOTA) 달성
    • GPT-4V, Claude 3.5 Sonnet, Gemini Pro 1.5 등과 비교하여 더 나은 성능을 발휘
    • 다양한 데이터셋에서 학습되어 높은 정확도와 성능을 보장
    • Hugging Face에서 무료 API 제공, 누구나 사용할 수 있음
    • Gradio 데모를 통해 사용자들이 쉽게 테스트하고 활용 가능
    • 다중 모드 챗봇 기능을 제공하여 사용자와의 상호작용 향상
